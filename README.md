# RAWR AI Consulting

Rethink HR. Powered by AI.

## Overview

RAWR AI Consulting is a fictional AI-powered HR consulting company created as part of a university project in the course *Practical Application of Online Marketing* at Heilbronn University.

The project focuses on the integration of artificial intelligence into modern HR processes, including recruiting, process automation, predictive analytics, and digital HR assistants.

This website and all related content are entirely fictional and created exclusively for educational purposes.

---

## Features

- Modern landing page for an AI HR consulting company
- Responsive design
- LinkedIn integration and social proof elements
- Service and process presentation
- Legal pages (Impressum & Datenschutz)
- Built with React, TypeScript, Vite, Tailwind CSS and shadcn/ui

---

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Bun / npm

---

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
